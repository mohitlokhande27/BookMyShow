# BookMyShow - Spring Boot Application

A movie booking system built with Spring Boot, JPA, and MySQL.

## Features
- Movie management
- Show scheduling
- Seat booking
- Payment processing

## Technologies Used
- Java 17
- Spring Boot 3.5.7
- Spring Data JPA
- MySQL
- Lombok

## Setup
1. Clone the repository
2. Create MySQL database `bookmyshow`
3. Update `application.properties` with your MySQL credentials
4. Run the application
